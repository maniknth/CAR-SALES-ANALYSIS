# CAR-SALES-ANALYSIS
Power BI dashboard analyzing car sales data to generate business insights and trends
I worked on a Car Sales Analysis project using a dataset from Kaggle. The objective was to analyze sales performance and identify trends based on factors like region, car color, and body style.

First, I performed data cleaning by removing duplicates and handling missing values using Power Query in Power BI.

Then, I created a Calendar Table using DAX to extract year, month, and week from the sales date, which helped in time-based analysis. I also built proper data modeling by connecting the calendar table with the sales data.

Next, I created key KPIs such as:

YTD (Year-to-Date) Sales
PYTD (Previous Year-to-Date) Sales
MTD (Month-to-Date) Sales
Average Sales
and compared YTD vs PYTD to measure growth

Finally, I built an interactive dashboard using:

Line chart for monthly sales trends
Donut charts for sales by color and body style
Slicers for filters like transmission, color, and body type

This dashboard helped in identifying sales patterns, top-performing categories, and overall business performance.”
